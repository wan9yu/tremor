tremor

An honest, long-running instrument for one question: is the world actually getting more disordered — or does it only feel that way?

– / 4 lines trembling today

Tension lines, normalized

Each line on its own robust z-score, so different units share one axis. Dashed band = the ±3 trembling threshold.

tremor doesn't predict the end — nobody can. It only makes now readable. It's a seismograph, not a prophet, and it's blind to instantaneous shocks; it listens for the slow, systemic, trace-leaving kind of disorder.

Data is collected daily by a GitHub Actions cron and committed back to the repo (git-scraping). Charts read data/*.csv directly — no backend.
